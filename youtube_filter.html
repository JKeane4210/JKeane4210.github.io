MyYoutubeFilter

Having a lot of activities that I am juggling on regular basis between school, soccer, competitive programming, and personal projects, I need to make sure that I have the right environment to set me up for success in a given day. In my freshman year of college, I started using the screen time widget, which showed me how much time I was wasting every day watching YouTube. I considered blocking my access to the site, but I realized that my time on YouTube was not all bad. Even further, the content I relaxed to was not all bad either, as I did want to follow certain content creators.

Because of this, I realized that my gripes with YouTube came mostly down to the "For You" page. The Youtube algorithm was offering me videos that were related to the channels that I liked, but they were not truly what I was looking for. But because accessing these videos was so easy, I would end up wasting time on the "For You" page that I regretted later. To solve this, over Thanksgiving and Winter break of 2021, I built an application using node-webkit and NodeJS to be an interface into the Youtube API that would only provide me with videos that are new from my current subscribers, successfully filtering out the content that was distracting me when I visited YouTube.

Project Analysis:

What did you do well in the project?

I did well in recognizing a problem and looking for a solution, even if one did not currently exist. I did well at applying my previous knowledge of productivity habits to try and create a technical solution to the distractions YouTube provided. Finally, I did well at maintaining consistent progress on this personal project during the course of the school year.

What new skills/knowledge did you gain from working on the project? What old skills/knowledge did you revisit or improve?

In this project, I learned how to interact with a google API, specifically the YouTube Data API. Additionally, to get the specific subscribers of a user, I learned how to authenticate users using OAuth2.

In creating the GUI, I had previously learned how to create Desktop applications from HTML and CSS in NodeJS using the node-webkit framework. I applied my previous understanding of HTML/CSS and NodeJS to create this new application.
