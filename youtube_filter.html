MyYoutubeFilter

Fall & Winter 2021

Role: Developer

Type: Personal Project

Team: Individual

Description:

Having a lot of activities that I am juggling, I need to make sure that I have the right environment to set me up for success. In my first year of college, I started using the screen time widget. Using this tool showed me how much time I wasted every day watching YouTube. I realized that my gripes with YouTube came down to the "For You" page. The Youtube algorithm offered me videos related to the channels I liked, but it made getting this content too easy, distracting me too often.

To solve this, during the fall and winter of 2021, I built an application using node-webkit and NodeJS that gave me my YouTube feed through the Youtube API. However, this tool would only provide me with new videos from my current subscribers, successfully filtering out the distracting content.

Project Analysis:

What Went Well:

  • Applying Knowledge

    From my readings on productivity in books such as Atomic Habits by James Clear, I recognized that my bad habit was triggered by how obvious YouTube thumbnails were. By removing the thumbnails and trimming the number of videos displayed, I created an application to help me.

  • Recognizing Problems/Opportunities

    I did well in recognizing a problem and looking for a solution, even if one did not currently exist. Additionally, I recognized that every part of the problem with YouTube was digital, making it feasible to address this through programming.

  • Improvement On Project Management

    Compared to my LinkedIn messenger, I created this application in much less time. The product may have had less functionality needed for a prototype, but regardless, I feel like I managed this project with more direction. I did this by sticking to my scrum boards more religiously.

What Could Be Improved:

  • Sharing My Work

    Additionally, as I have shared this product with others, many have said that I should publish this application because many want to improve their productivity. This project could become a product if I get it into the user's hands.

Skills Used/Learned:

What I Learned:
  • Google APIs & OAuth2 Authentication

    In this project, I learned how to interact with a Google API. Specifically, I used the YouTube Data API. Additionally, to get the specific subscribers of a user, I learned how to authenticate users using OAuth2.

What I Improved/Reused:
  • Node Webkit

    In creating the GUI, I had previously learned how to create Desktop applications from HTML and CSS in NodeJS using the node-webkit framework. I applied my previous understanding of HTML/CSS and NodeJS to create this new application.
